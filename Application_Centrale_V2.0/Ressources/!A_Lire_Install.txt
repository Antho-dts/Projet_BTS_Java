1- Dans le projet Angular Connexion au server.js d'acces à la BDD mysql avec websocket
		accueil.component.ts
		ligne 33: this.websocketService.connect('ws://127.0.0.1:3000');
		Definir l'adresse du poste ou se trouve le server.js donc le projet Angular!
		
2- Dans le serveur websocket server.js
		Angular_Capteur_Simplifie\mon-projet-nodejs-rias\server.js	
		ligne 18: database: 'mysql'
		Définir le nom de la base SQL ici 'mysql' et non 'base_rias'
		
3- Demarrage du server.js 
		dans une console
		Angular_Capteur_Simplifie\mon-projet-nodejs-rias>node server.js
		
		Retour console:
			Serveur Node.js démarré sur http://localhost:3000
			Connecté à la base de données MySQL
		
4- Demarrer le site Angular
